MDTSublimados

MDT Sublimados – Sitio Web Comercial Desarrollo full-stack de e-commerce para emprendimiento local Diseñé y desarrollé desde cero la plataforma web de MDT Sublimados, un emprendimiento de sublimación y productos personalizados ubicado en Tristán Suárez, Buenos Aires. El sitio combina una experiencia de usuario intuitiva con funcionalidades comerciales robustas, pensado especialmente para emprendedores que necesitan una solución ágil y efectiva para gestionar pedidos y mostrar sus productos. Tecnologías utilizadas: Frontend: HTML5, CSS3, JavaScript (responsive y mobile-first) Backend & Base de datos: Firebase (Firestore, Authentication, Cloud Storage) Integraciones: WhatsApp para finalización de pedidos, Google Analytics, Google Maps Herramientas: CSV para importación masiva de productos, panel de administración con autenticación segura

Mi Nuevo Proyecto

Descripción

MDT Sublimados – Sitio Web Comercial Desarrollo full-stack de e-commerce para emprendimiento local Diseñé y desarrollé desde cero la plataforma web de MDT Sublimados, un emprendimiento de sublimación y productos personalizados ubicado en Tristán Suárez, Buenos Aires. El sitio combina una experiencia de usuario intuitiva con funcionalidades comerciales robustas, pensado especialmente para emprendedores que necesitan una solución ágil y efectiva para gestionar pedidos y mostrar sus productos. Tecnologías utilizadas: Frontend: HTML5, CSS3, JavaScript (responsive y mobile-first) Backend & Base de datos: Firebase (Firestore, Authentication, Cloud Storage) Integraciones: WhatsApp para finalización de pedidos, Google Analytics, Google Maps Herramientas: CSV para importación masiva de productos, panel de administración con autenticación segura Funcionalidades clave: ✔ Catálogo dinámico con productos destacados ✔ Sistema de gestión de productos (CRUD completo + importación por CSV) ✔ Formulario de contacto con validación ✔ Footer interactivo con mapa y redes sociales ✔ Carrito de compras orientado a conversión por WhatsApp ✔ Diseño 100% responsive y optimizado para móviles El proyecto refleja mi enfoque en soluciones prácticas, escalables y centradas en el usuario, priorizando la usabilidad para clientes no técnicos y la eficiencia operativa para el administrador.

Tecnologías

  • HTML5
  • Tailwind CSS
  • JavaScript
  • Responsive Design
